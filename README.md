# AI Customer Churn Prediction 

## Overview

This project is an end-to-end machine learning and analytics system designed to predict customer churn and provide actionable business insights through an interactive dashboard.

It combines predictive modeling with data visualization to help organizations identify at-risk customers, understand churn drivers, and make data-driven retention decisions.

---

## Key Features

* End-to-end ML pipeline (data preprocessing → training → evaluation → prediction)
* Interactive dashboard for business insights
* Customer churn risk classification (Low / Medium / High)
* Feature importance analysis to identify key churn drivers
* Data visualization for trends and patterns
* Model persistence for reuse and deployment

---

## Business Value

This system is not limited to prediction—it enables decision-making:

* Identifies high-risk customers for targeted retention campaigns
* Highlights factors contributing to churn (e.g., contract type, tenure, services)
* Supports strategic planning using data-backed insights
* Reduces customer acquisition cost by improving retention
